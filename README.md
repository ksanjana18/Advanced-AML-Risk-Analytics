# 📖 About the Project

The **Advanced AML Risk Analytics Dashboard** is a Machine Learning-based web application that detects suspicious financial transactions and helps identify potential Anti-Money Laundering (AML) activities.

Built with **Python**, **Streamlit**, **Pandas**, **Plotly**, and **Scikit-learn**, the dashboard provides interactive analytics, anomaly detection, and risk classification for banking transactions.
